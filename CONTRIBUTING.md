# Contributing to Early Frontier

Thanks for helping improve Early Frontier! This project documents practical AI
agent workflows with clear steps, inputs, and expected outputs.

## Quick start

```bash
bun install
bun run dev
```

Open http://localhost:3001

## Ways to contribute

- Add or improve a workflow with steps, inputs, and expected outputs
- Update guides when tools or behavior change
- Share concrete examples, templates, or screenshots
- Fix typos, broken links, or unclear sections

## Content guidelines

- Keep guidance practical and testable
- Include prerequisites, inputs, and expected outputs
- Prefer short steps over long paragraphs
- Call out safety or permissions concerns when relevant

## Submitting changes

1. Create a branch for your work.
2. Make changes and verify locally if possible.
3. Open a PR with a short summary and any context or screenshots.

If you’re unsure where to start, open an issue with the change you’d like to make
and the page it affects.
